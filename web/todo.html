5 · Texture Pixelation Workflow (tooling ready)

For real photographic textures (rock, bark, dirt, snow) we bake pixel versions offline — never at runtime — exactly as requested. The tool is committed at web/scripts/pixelate.js:

  1. npm i -D sharp (one-time)
  2. Drop source photos in web/raw/
  3. npm run pixelate -- --in raw/granite.jpg --out public/assets/tex/granite.png --size 128 --quantize

--size sets the pixel grid resolution, --quantize snaps colors to the game palette, output is upscaled nearest-neighbour so the committed asset stays crisp. Commit the baked PNG, not the raw photo.

Because pixel-art games generally avoid tiled textures, prefer this only for: ground/dirt strips, rock faces on near-ridge layers, and town building facades. Most surfaces should stay flat-shaded.
